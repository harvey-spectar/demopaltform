# Quiz Platform Starter

A simple starter quiz platform (HTML/CSS/JS).

## Run

Open `index.html` directly, or run:

```bash
cd quiz-platform
python3 -m http.server 8787
```

Then visit `http://127.0.0.1:8787`.
